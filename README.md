# LibreNMS Device Photo Plugin

A LibreNMS plugin that adds device photos to the device overview page.

## Features

- Shows device photos on the LibreNMS device overview page
- Thumbnail view
- Click to zoom/open larger image
- Admin-only upload/delete functionality

## Notes

This repository should only contain the plugin code.

Do not commit:
- LibreNMS config.php
- .env files
- API keys
- passwords
- internal customer data
- real photos of internal equipment
